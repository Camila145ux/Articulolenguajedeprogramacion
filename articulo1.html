HTML

(siglas de HyperText Markup Language, en español Lenguaje de Marcado de Hipertexto) es el lenguaje estándar que se utiliza para crear y estructurar páginas web. No es un lenguaje de programación, sino un lenguaje de marcado, lo que significa que se utiliza para "marcar" el contenido con etiquetas que le indican al navegador cómo mostrarlo.

Caracteristicas

Trabaja en conjunto con CSS (para estilos) y JavaScript (para interactividad).
